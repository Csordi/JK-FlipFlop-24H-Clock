# JK-FlipFlop-24H-Clock
24H TTL Clock only with TTL integrated circuits.

- 24H clock (no AM/PM mode)
- Time format: HH:MM:SS
- Red 7segment displays
- 6-18V DC IN
- Almost fully through-hole design (only one component is SMD)
